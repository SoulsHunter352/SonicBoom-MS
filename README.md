# SonicBoom-MS
SonicBoom — музыкальный сервис для поиска и прослушивания музыки на любой вкус. Удобный интерфейс для авторов, позволяющий быстро и легко добавить свои музыкальные плейлисты. Обычному же пользователю открывается широкий спектр жанров и тегов, с помощью которых каждый найдёт музыку по себе.
